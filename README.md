# 🍷 Wine Quality EDA

This project explores the **Wine Quality Dataset** through exploratory data analysis (EDA) using Python.

## 📊 Overview
The analysis covers:
- Data cleaning and preprocessing
- Statistical summaries
- Correlation analysis
- Visualization of key features

## 🧠 Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
